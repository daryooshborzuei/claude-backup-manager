🔒 Privacy: all processing happens locally in your browser. Nothing is uploaded anywhere — you can even disconnect from the internet after this page has loaded once.
What a Claude export contains: conversations.json (all chats with full messages, attachments & tool activity), users.json (account profile), memories.json (account-wide memory + per-project memories, included in recent exports), and a projects/ folder (one JSON per project with instructions and documents). Binary files (images, original PDFs) are not included in Claude exports — only their extracted text. If Anthropic sent you a .dms file, it is a renamed ZIP and can be loaded here directly. Large exports may arrive as several batch-000N.zip files; load each one separately.
